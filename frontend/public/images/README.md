## How to Add Images

1. Upload your images to this `public/images/` directory
2. Use the exact filenames listed above, or update the code to match your filenames
3. Supported formats: JPG, PNG, WebP
4. Recommended sizes:
   - Feature images: 800x600px
   - Testimonial avatars: 200x200px (square)
   - Step images: 600x400px
   - Hero image: 1200x800px
