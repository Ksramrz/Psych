# ClinicSense Landing Page - Figma Design Brief

## Project
Design a modern, professional landing page for **ClinicSense** - AI-powered platform for psychologists. Light theme, clean, trustworthy, innovative.

**Target**: Licensed psychologists, therapists, mental health professionals
**Tone**: Professional, trustworthy, innovative
**Colors**: Light theme with blue/cyan gradients (avoid dark backgrounds)

## Required Sections

### 1. Navigation
- Logo: "ClinicSense" (text or icon+text)
- Right: "Sign In" link + "Get Started Free" button
- Sticky header, light background, subtle shadow

### 2. Hero
- Headline: "Save Hours on Case Documentation. Focus on What Matters."
- Subheadline: AI-powered documentation, case analysis, ethics checks, supervisor support
- CTAs: Primary "Try it Free Now" (gradient), Secondary "Sign In" (outline)
- Trust badge: "Trusted by psychologists nationwide"
- Visual: Subtle illustration/gradient/product mockup

### 3. Testimonials
- Title: "What Psychologists Are Saying"
- 3 cards: Quote, author name, role/credentials
- Subtle shadows, hover effects

### 4. Core Features (4 cards grid)
1. **AI Case Analysis** - Instant insights, risk assessments, evidence-based recommendations
2. **Smart Note Summarization** - Transform raw notes into structured documentation
3. **Ethics & Compliance** - Built-in checklists and compliance verification
4. **Supervisor Reflection** - AI-powered supervision support

### 5. Extended Features (6-9 cards grid, present as existing)
5. Voice Input & Dictation - Real-time transcription

7. EHR Integration - SimplePractice, TherapyNotes integration
8. Client Management - Profiles, session history, progress tracking

### 7. Final CTA
- Gradient background (blue to cyan)

### 8. Footer
- 4 columns: Branding, Product links, Resources, Contact
- Copyright notice

## Design Requirements

**Visual**: Light, clean, modern. White→light gray/blue gradient backgrounds. White cards with subtle shadows, rounded corners. Clean typography, generous whitespace.

**Colors**: Primary Blue (#2563eb) to Cyan (#06b6d4) gradients. Text: dark gray headings, medium gray body. Backgrounds: white, #f9fafb, #eff6ff. Avoid dark backgrounds.

**Interactions**: Button hover (scale, shadow, color shift). Card hover effects. Smooth transitions.

**Responsive**: Desktop primary (1920px/1440px). Mobile: stacked cards, adjusted spacing. Grid: 4→2→1 columns.

## Inspiration
Modern SaaS (Stripe, Linear, Vercel). Healthcare software (modern SimplePractice/TherapyNotes). AI/tech products. Avoid: clinical aesthetic, dark themes, clutter.

## Deliverables
1. Desktop design (1920px/1440px)
2. Component specs: buttons, cards, typography scale, color palette (hex codes), spacing system
3. Optional: Mobile layout (375px), hover states

## Notes
- All features presented as **existing**, not "coming soon"
- Premium, trustworthy feel
- Clean, professional, scannable
- Subtle AI-themed elements (minimal)
- Key message: "Complete AI-powered platform helping psychologists save time, reduce risk, improve outcomes - all in one solution"
